Shader uniform extraction

The example demonstrates also the new osgUtil.ShaderParameterVisitor.js . This visitor extract uniform and insert sliders in you page to customize value. It's useful to fine tune shader effect. It also saves the result into the localStorage for conveniance. The visitor let you redefine parameters for example if you want to change the range. 

    // add a comment like the following line into your shader to configure the slider
    // "uniform float density; //  { "min": 0.0,  "max": 0.006, "step": 0.001, "value": 0.00001 } ", 

    var params = new osgUtil.ParameterVisitor();
    params.setTargetHTML(document.getElementById('Parameters'));
    group.accept(params);
